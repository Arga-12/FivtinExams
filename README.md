# FivtinExams
FivtinExams adalah aplikasi ujian berbasis Flutter yang dirancang khusus untuk SMPN 15 Malang. Aplikasi ini bertujuan untuk mempermudah siswa dalam mengakses ujian dan materi belajar secara digital.
